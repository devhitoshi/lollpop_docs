---
1. ユーザーから提供された画像やテキストから歌詞を転記します。
2. 構成（イントロ、Aメロ、Bメロ、サビ、間奏、ブリッジ、Cメロ、落ちサビ、アウトロなど）を推測、または既存のコール表などを参考に特定します。
3. 作詞、作曲、レコーディングエンジニアなどの情報を、公式リンク集（`link.md`）にある TuneCore や Apple Music のウェブサイトから `read_url_content` を使って取得します。
4. 以下の形式で、プロジェクト内の `work/` ディレクトリに曲名に基づいたファイル名（例：`[曲名]_歌詞.md`）で保存します。
    - `# 曲名`
    - `## 構成名`
    - `### 歌詞`
    - `(構成ごとに繰り返し)`
    - `## 作詞 作曲`（作詞、作曲、エンジニアなどの情報を記載）
5. 作成後、ユーザーにレビューを依頼し、必要に応じて構成の名称や歌詞の細部を調整します。
