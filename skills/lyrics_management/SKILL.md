# Lyrics Management Skill

このスキルは、「ろりぽっぷ!!!!!!!」の楽曲歌詞ドキュメントを、指定されたフォーマットに従って作成・管理するために使用します。

## 手順
1. **歌詞の提供**: ユーザーから画像（OCRなどで読み取る）またはテキスト形式の歌詞を提供されます。
2. **構成（セクション）の整理**:
   - 歌詞の流れから構成（イントロ、Aメロ、Bメロ、サビ、間奏、Cメロ、落ちサビ、アウトロなど）を推測、または既存のコール表を参考に定義します。
3. **クレジット情報の取得**:
   - `link.md` にある各配信サイト（TuneCore, Apple Music等）のリンクから `read_url_content` を実行し、**作詞家、作曲家、エンジニア**などのクレジット情報を詳細に取得します。
4. **Markdownドキュメントの作成**: 以下の形式で構成します。
    - `# 曲名`
    - `## 構成名`
    - `### 歌詞`
    - `(構成ごとに繰り返し)`
    - `## 作詞 作曲`（作詞、作曲、エンジニアなどの情報を記載）
5. **保存**:
   - 作成したファイルをプロジェクト内の `songs/lyrics/` ディレクトリに、`[曲名].md` というファイル名で保存します。

## 利用可能なリソース
- **`link.md`**: 公式配信サイトのリンクが記載されています。
- **`songs/call_list.md`**: 既存曲の構成やコールの情報が含まれています。
- **`songs/lyrics/`**: 歌詞ドキュメントの最終的な保存先です。
