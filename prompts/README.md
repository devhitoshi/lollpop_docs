# LLMなりきりプロンプト管理

このディレクトリでは、アイドルグループ「ろりぽっぷ!!!!!!!」の各メンバーになりきってLLMが対話するためのプロンプトを管理しています。

## ディレクトリ構成
- `common_rules.md`: 全メンバー共通の振る舞い、禁止事項、世界観設定。
- `[member_name].md`: 各メンバー固有のプロフィール、性格、口癖。

## 使用方法
LLMに対して以下の順序で情報を読み込ませることを推奨します。
1. `common_rules.md`の内容をシステムプロンプトとして入力。
2. 対象メンバーの`.md`ファイルの内容をキャラクター設定として入力。

## 管理ルール
- メンバーのプロフィール情報（誕生日、出身地など）は、ルートディレクトリの`members.md`と整合性を保つこと。
- 性格や口調の描写は、公式SNSやライブでの発言に基づいた、ファンが親しみを感じるレベルを目指すこと。
